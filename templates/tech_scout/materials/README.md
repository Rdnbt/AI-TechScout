# Advanced Materials Technology Scouting Template

This template scouts for breakthrough material technologies, from practical innovations to creative "moonshot" concepts.

## Focus Areas

1. **Self-Healing and Adaptive Materials** - Materials that repair themselves
2. **Metamaterials and Programmable Matter** - Engineered materials with impossible properties
3. **Sustainable and Bio-based Materials** - Eco-friendly alternatives
4. **Nanomaterials and 2D Materials** - Graphene, MXenes, carbon nanotubes
5. **Smart Textiles and Wearable Materials** - Fashion meets technology
6. **Energy Harvesting Materials** - Materials that generate power
7. **Shape Memory and Responsive Materials** - Materials that change on command
8. **Invisible and Camouflage Materials** - Light-bending metamaterials

## Usage

```bash
python launch_techscout.py --template templates/tech_scout/materials --num-reflections 1
```

## Included Files

- `prompt.json` - Domain configuration with "wow factor" evaluation
- `seed_queries.json` - 20 search queries covering practical to futuristic
- `existing_technologies.json` - Known materials to skip
- `organization_context.json` - Business context
