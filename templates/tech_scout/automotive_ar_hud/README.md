# Automotive AR Head-Up Display Template

## Overview

This template is designed for scouting emerging technologies in **Augmented Reality Head-Up Displays (AR HUDs)** for automotive windshields. It targets both incremental HUD improvements and breakthrough full-windshield AR projection systems.

## Target Technologies

- **Holographic Waveguide Displays** - Companies like Envisics, WayRay
- **Full-Windshield AR** - Projection across entire windshield surface
- **MicroLED/MicroOLED** - Next-gen display sources for HUDs
- **Laser Beam Scanning** - MEMS-based projection systems
- **Light Field Displays** - True 3D depth without eye strain

## Use Case Scenarios

1. **AR Navigation** - Turn-by-turn overlaid on actual road
2. **ADAS Visualization** - Highlight pedestrians, lane markings, hazards
3. **Night Vision Enhancement** - Thermal/IR overlay on windshield
4. **Parking Assistance** - AR guidelines and obstacle highlighting
5. **Infotainment** - Passenger-side content (dual-view displays)

## Regulatory Considerations

- FMVSS 111 (US) - Rear visibility
- ECE R46 (EU) - Field of vision
- Driver distraction guidelines (NHTSA, Euro NCAP)
- Windshield optical distortion limits
